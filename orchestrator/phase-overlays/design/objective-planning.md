# Design Objective Planning Overlay

Allowed planning: define the minimum capability lanes, dependency edges, and handoff structure needed to produce approved contracts, interfaces, and design artifacts for the objective.

This prompt is for planning only. Do not describe implementation, file edits, or command execution as current allowed work.

Keep the outline lean. Prefer the smallest lane set that can close the required design surface.

Forbidden planning: implementation work that is not strictly required to validate a design artifact, shell commands, and evidence-only lanes.
